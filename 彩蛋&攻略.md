## 系统特性（重制版新增）

| 特性 | 说明 |
|------|------|
| **Toast通知系统** | 替代 alert，更沉浸，支持 lore/ok/err/warn 类型 |
| **任务栏最小化** | 窗口可最小化到任务栏，带活跃状态指示 |
| **环境音开关** | 右下角 🔇/🔊 按钮，Aero阶段变为水下音效 |
| **存档系统** | localStorage 自动保存进度，游戏结束清除 |
| **图片加载反馈** | 明确显示成功/失败状态，含尺寸信息 |
| **format C: 确认** | 验证码机制防止误触 |
| **音效系统** | Web Audio API 合成音效，无需外部文件 |
| **白色文字** | 第二阶段桌面图标自动变为白色+深色阴影 |
